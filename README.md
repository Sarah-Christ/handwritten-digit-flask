# 🖐️ Handwritten Digit Recognizer Web App

A simple Flask-based web app that uses a Convolutional Neural Network (CNN) to recognize handwritten digits (0–9) drawn on a canvas, trained on the MNIST dataset.

---

## 📸 Demo

Draw a digit on the canvas, click **Predict**, and get the recognized digit with confidence.

---

## 🚀 Features

- 🧠 Trained on MNIST dataset using TensorFlow/Keras
- 🎨 Frontend canvas to draw digits
- 🔍 Real-time digit prediction via Flask backend
- 📷 Preprocessing: grayscale, invert, resize, reshape
- ✅ Displays predicted digit with confidence score
